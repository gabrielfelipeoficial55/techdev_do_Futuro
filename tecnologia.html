Tecnologias

HTML

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Ele permite a construção de estruturas de conteúdo na web, como textos, imagens, links e outros elementos multimídia. HTML é fácil de aprender e é a base de todas as páginas da internet.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de documentos escritos em HTML ou XML. Com o CSS, é possível controlar o layout, cores, fontes e espaçamento dos elementos, experiência visual mais atraente e consistente para os usuários.

JS

JavaScript é uma linguagem de programação essencial para o desenvolvimento web, permitindo criar páginas interativas e dinâmicas. Introduzida em 1995, é amplamente utilizada tanto no frontend quanto no backend (com Node.js). Suas principais características incluem manipulação do DOM, resposta a eventos e suporte a APIs.

React

React é uma biblioteca JavaScript de código aberto para construir interfaces de usuário, mantida pelo Facebook e por uma comunidade. Permite criar componentes reutilizáveis que gerenciam seu próprio estado, resultando em interfaces eficientes e fáceis de manter. É amplamente utilizada para aplicações web modernas e dinâmicas.

Node.js

Node.js é um ambiente de execução JavaScript no lado do servidor, criado em 2009 por Ryan Dahl. Ele permite a construção de aplicações escaláveis e de alta performance, usando um modelo de I/O não-bloqueante e orientado a eventos. Amplamente utilizado para desenvolvimento backend, APIs e aplicações em tempo real.

PHP

PHP é uma linguagem de script open-source voltada para o desenvolvimento web. Criada por Rasmus Lerdorf em 1994, permite a criação de páginas dinâmicas e interativas. Executada no servidor, é amplamente usada para integrar com bancos de dados e construir aplicações web robustas e eficientes.
